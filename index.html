FeatureRefiner

A no-code solution for performing data transformations like imputation, encoding, scaling, and feature creation, with an intuitive interface for interactive DataFrame manipulation and easy CSV export.

Key Features

Upload Data

Quickly and easily upload your CSV files directly into the application for immediate access.

Data Transformations

Choose from a variety of transformation options to tailor your data preprocessing.

Data Visualization

Use an interactive grid to visualize your DataFrame and inspect transformations.

Download Transformed Data

Export the modified DataFrame as a CSV file for further analysis or integration.

Installation and Usage

To install FeatureRefiner in an isolated environment, follow these steps:

  1. Create a virtual environment: 
    python -m venv feature_refiner_env
  2. Activate the virtual environment:
    • On Linux/macOS: 
      source feature_refiner_env/bin/activate
    • On Windows: 
      feature_refiner_env\Scripts\activate
  3. Install FeatureRefiner: 
    pip install FeatureRefiner
  4. Run the package: 
    run-FeatureRefiner
