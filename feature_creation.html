Feature Creation

The PolynomialFeaturesTransformer class generates polynomial features from the numeric columns of a DataFrame. This transformation allows models to capture interactions between features and can enhance the predictive power of certain algorithms.

Polynomial Feature Creation

Example Use Case:

  1. Transforming a DataFrame with Polynomial Features
    2. import pandas as pd
from FeatureRefiner.create_features import PolynomialFeaturesTransformer

# Create a sample DataFrame
data = {
    'height': [5.5, 6.0, 5.8],
    'weight': [150, 180, 160]
}
df = pd.DataFrame(data)

# Initialize the PolynomialFeaturesTransformer with degree 2
transformer = PolynomialFeaturesTransformer(degree=2)

# Transform the DataFrame to include polynomial features
transformed_df = transformer.fit_transform(df)
print(transformed_df)
  2. Updating the Polynomial Degree

    3. You can also update the polynomial degree after initialization and transform the data again.

    # Update the polynomial degree to 3 and transform again
transformed_df_updated = transformer.fit_transform(df, degree=3)
print(transformed_df_updated)
