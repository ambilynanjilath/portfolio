DateTime Data Handling

The DateTimeExtractor class is designed to extract and parse datetime components from a specified column in a pandas DataFrame. This class facilitates the manipulation of datetime data, making it suitable for further analysis or modeling.

Example:

import pandas as pd
from FeatureRefiner.datetime_features import DateTimeExtractor

# Create a sample DataFrame
data = {'date_column': ['01/15/2023 10:30', '2023-01-16T15:45:00', 'invalid_date']}
df = pd.DataFrame(data)

# Initialize the DateTimeExtractor
extractor = DateTimeExtractor(df, 'date_column')

Date Extracting

_parse_date

This private method attempts to parse a date string using multiple formats. If parsing fails for all formats, it logs a debug message and returns pd.NaT.

Example:

parsed_date = extractor._parse_date('01/15/2023 10:30')

Year Extracting

extract_year

This method extracts the year from the datetime column and adds it as a new column named 'year'.

Returns:

  • pandas.DataFrame: DataFrame with an added 'year' column.

Raises:

  • ValueError: If there is an error extracting the year.

Example:

df_with_year = extractor.extract_year()

All Extracting

extract_all

This method extracts year, month, day, and day of the week from the datetime column and adds them as new columns.

Returns:

  • pandas.DataFrame: DataFrame with added 'year', 'month', 'day', and 'day_of_week' columns.

Raises:

  • ValueError: If there is an error extracting any of the datetime components.

Example:

df_with_all = extractor.extract_all()
