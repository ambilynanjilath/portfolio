Missing Value Imputation

The MissingValueImputation class provides several strategies to handle missing values in a dataset, such as:

  • Mean: Replaces missing values with the column's mean.
  • Median: Replaces missing values with the column's median.
  • Mode: Replaces missing values with the column's most frequent value.
  • Custom Value: Allows you to specify a custom value to fill in missing data.

Mean Imputation

Mean imputation fills missing values with the average of the non-missing values in a numeric column.

Use Cases:

  • Suitable for normally distributed numerical data.
  • Effective when the dataset does not have significant outliers.

Example:

import pandas as pd
import numpy as np
from FeatureRefiner.imputation import MissingValueImputation

# Sample DataFrame with missing values
df = pd.DataFrame({'A': [1, 2, np.nan, 4, 5]})
imputer = MissingValueImputation(strategies={'A': 'mean'})
imputed_df = imputer.fit_transform(df)
print(imputed_df)

Median Imputation

Replaces missing values with the median of the non-missing values in a numeric column.

Use Cases:

  • Ideal for numeric data with outliers, as the median is less sensitive to extreme values.
  • Useful when the data is skewed.

Example:

# Sample DataFrame with missing values
df = pd.DataFrame({'B': [1, 2, np.nan, 4, 100]})
imputer = MissingValueImputation(strategies={'B': 'median'})
imputed_df = imputer.fit_transform(df)
print(imputed_df)

Mode Imputation

Fills missing values with the most frequent value in a categorical or discrete numeric column.

Use Cases:

  • Suitable for categorical data where the most common category is a meaningful replacement.
  • Effective for numeric data if the data is not normally distributed.

Example:

# Sample DataFrame with missing values
df = pd.DataFrame({'C': ['cat', 'dog', np.nan, 'cat', 'dog']})
imputer = MissingValueImputation(strategies={'C': 'mode'})
imputed_df = imputer.fit_transform(df)
print(imputed_df)

Custom Value Imputation

This method allows you to specify a custom value to replace missing entries in a column.

Use Cases:

  • Useful when a specific value is meaningful for the context of the data.
  • Helps when you want to avoid the bias introduced by mean/median/mode.

Example:

# Sample DataFrame with missing values
df = pd.DataFrame({'D': [1, 2, np.nan, 4, 5]})
imputer = MissingValueImputation(strategies={'D': 0})  # Custom value of 0
imputed_df = imputer.fit_transform(df)
print(imputed_df)
